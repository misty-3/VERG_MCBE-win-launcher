# 🇬🇧 English

A powerful, modern desktop application designed to easily download, manage, and install various versions of Minecraft Bedrock Edition for Windows (both UWP and GDK). Manage your installations effortlessly, enable developer mode with a click, and explore both Release and Preview builds!

## 🌟 Features
- **Download & Install**: Fetch officially archived UWP/GDK versions of Minecraft Bedrock Edition.
- **Version Management**: View installed versions, import custom versions, and keep your files organized.
- **One-Click Setup**: Built-in checkers and toggles for Windows Developer Mode and Decryption Keys.
- **Multilingual Support**: Fully supports English and Arabic interfaces out of the box.
- **Modern UI**: Clean and responsive WPF-based UI featuring filtering, auto-scrolling, and beautiful aesthetics.

## 🚀 Getting Started
1. **Prerequisites**: Windows 10 or Windows 11. Ensure Developer Mode is enabled (the app has a built-in feature to help you enable this if you haven't already!).
2. **Build**: Open `MCLauncher.sln` in Visual Studio and build the solution (requires .NET Framework 4.8).
3. **Run**: Launch the application. You can easily switch between languages (English/Arabic) in the top right corner.

## 🛠 Tech Stack
- **C# / WPF**: Frontend UI and App logic.
- **.NET Framework 4.8**: Core framework.
- **Newtonsoft.Json**: Serialization/Deserialization.

# 🇸🇦 العربية (Arabic)

تطبيق سطح مكتب قوي وحديث مصمم لتنزيل وإدارة وتثبيت إصدارات مختلفة من لعبة ماينكرافت بيدروك (Minecraft Bedrock Edition) لنظام التشغيل ويندوز (UWP و GDK) بسهولة. يمكنك إدارة تثبيتاتك بكل سهولة، وتفعيل وضع المطور بضغطة زر، واستكشاف كل من الإصدارات النهائية (Release) والتجريبية (Preview)!

## 🌟 الميزات
- **التنزيل والتثبيت**: جلب الإصدارات الرسمية المؤرشفة (UWP/GDK) من ماينكرافت بيدروك.
- **إدارة الإصدارات**: عرض الإصدارات المثبتة، استيراد إصدارات مخصصة خارجية، والحفاظ على تنظيم ملفاتك.
- **إعداد بنقرة واحدة**: أدوات مدمجة للتحقق من وتفعيل وضع مطور ويندوز (Developer Mode) ومفاتيح فك التشفير (Decryption Keys).
- **دعم متعدد اللغات**: يدعم الواجهات باللغتين الإنجليزية والعربية بالكامل وبشكل مدمج.
- **واجهة مستخدم حديثة**: لغة تصميم WPF نظيفة وسريعة الاستجابة تتميز بالفلترة، التمرير التلقائي، والجماليات البصرية المعاصرة.

## 🚀 البدء
1. **المتطلبات الأساسية**: ويندوز 10 أو ويندوز 11. تأكد من تفعيل "وضع المطور" (هناك ميزة مدمجة في التطبيق لمساعدتك في تفعيله في حال نسيانه!).
2. **البناء**: افتح `MCLauncher.sln` في Visual Studio وقم ببناء المشروع (يتطلب بيئة .NET Framework 4.8).
3. **التشغيل**: قم بتشغيل التطبيق المترجم. يمكنك التبديل بين اللغات (الإنجليزية/العربية) في الزاوية العلوية اليمنى.

## 🛠 التقنيات المستخدمة
- **C# / WPF**: واجهة المستخدم والمنطق البرمجي السحابي.
- **.NET Framework 4.8**: الإطار البرمجي الأساسي.
- **Newtonsoft.Json**: للتسلسل وإلغاء التسلسل للبيانات (JSON).
